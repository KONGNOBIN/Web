# EL
```

표현언어 (Expression Language)
1. 값을 표현하는 데 사용되는 스크립트 언어
2. JSP의 기본 문법을 보완하는 역할

EL 제공 기능
1. JSP 스코프에 맞는 속성 사용
2. 집합 객체에 대한 접근 방법 제공
3. 수치, 관계, 논리 연산자 제공
4. 자바 클래스 메소드 호출 기능 제공
5. EL만의 기본 객체 제공

```

## 문법
```
${expr} 
  * expr : 표현언어가 정의한 문법에 따라 값을 표현하는 식
```

## 접근 규칙
```

${<표현1>.<표현2>} 
  표현 1 or 표현 2 : null > null 반환.
  표현1 = Map일 경우 표현2를 key로한 값을 반환.
  표현1 = List or 배열이면 표현2가 정수일 경우 해당 정수 번째 index에 해당하는 값을 반환.
  <만약 정수가 아닐 경우에는 오류 발생>
  표현1이 객체일 경우 표현2에 해당하는 getter메소드에 해당하는 메소드를 호출한 결과를 반환.

```
